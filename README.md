# Building Production AI Pipelines

> **Track:** Anthropic Enterprise Implementation (Track 1)
> **Course 5 of 6**

Architect, orchestrate, and monitor production-grade AI pipelines with state management and cost controls.

---

## Modules

### Module 1: Pipeline Architecture Patterns

**Learning Objectives:**
- Choose between linear and DAG pipelines
- Design for idempotency and retries
- Handle long-running async tasks

### Module 2: Orchestration with LangChain/LlamaIndex

**Learning Objectives:**
- Build chains with LangChain
- Index and query documents with LlamaIndex
- Combine retrieval-augmented generation patterns

### Module 3: Memory, Context & State Management

**Learning Objectives:**
- Implement conversation memory
- Manage context window budgets
- Store and retrieve session state

### Module 4: Monitoring & Observability

**Learning Objectives:**
- Log token usage and costs
- Set up alerts for error spikes
- Build dashboards for pipeline health

### Module 5: Cost Optimization Strategies

**Learning Objectives:**
- Cache repeated prompts
- Select optimal models per task tier
- Implement prompt compression techniques
